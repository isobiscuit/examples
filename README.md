# Biscuit Examples
To build a biscuit use:
biscuit build {biscuit}
and run it:
biscuit run {biscuit}
- ## Basic
    - ### [Hello World](./basic/hello-world/)
        This Example will print "hello world" to the console
    - ### [Text2Text](./basic/text2text/)
        This Example will ask for a text and will print it
